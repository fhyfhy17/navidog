# 🐕 NaviDog

一个轻量级的 Web 端 MySQL 管理工具，风格类似 Navicat。

## 功能

- 🔗 **连接管理** — 支持 SSH 隧道、SSL 连接
- 📥 **NCX 导入/导出** — 兼容 Navicat 格式（含密码加密）
- 📊 **数据浏览** — 表格数据查看、筛选、分页
- 📝 **SQL 编辑器** — 语法高亮、自动补全、多结果集
- 📋 **表管理** — 复制表、删除表、清空表
- 💾 **数据导出** — 转储结构和数据为 SQL 文件
- 📄 **DDL 查看** — SQL 语法高亮
- 🖥️ **面板控制** — 左/右面板可折叠

## 快速开始

### 开发模式

```bash
# 安装依赖
npm install

# 启动开发服务（前端 + 后端同时启动）
npm run dev
```

浏览器打开 http://localhost:5173

### 生产部署

```bash
# 1. 安装依赖
npm install

# 2. 构建前端
npm run build

# 3. 启动服务
npm start
```

服务启动后打开 http://localhost:3001 即可使用。

### 自定义端口

```bash
PORT=8080 npm start
```

## 使用方式

1. 打开浏览器访问服务地址
2. 点击工具栏「连接」创建数据库连接
3. 填写主机、端口、用户名、密码等信息
4. 连接后左侧树可以浏览数据库和表
5. 双击表名查看数据，右键有更多操作
6. 点击「新建查询」打开 SQL 编辑器

## 技术栈

- **前端**: React + TypeScript + Vite + CodeMirror
- **后端**: Express + mysql2 + ssh2
- **部署**: 单进程同时服务 API 和前端静态文件
